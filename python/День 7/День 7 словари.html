Подробный разбор словарей (dict) в Python

Словари (dictionaries) — это одна из самых полезных и часто используемых структур данных в Python. Они хранят данные в формате пар “ключ: значение” и оптимизированы для быстрого поиска.

1. Зачем нужны словари?

Словари используют, когда нужно: - Хранить данные в структурированном виде (например, информацию о пользователе) - Быстро находить значения по ключу (например, телефонную книгу) - Подсчитывать количество повторяющихся элементов (частотный анализ) - Кэшировать результаты вычислений - Работать с JSON-данными (веб-API, базы данных)

Пример из реального мира:

user = {
    "name": "Анна",
    "age": 25,
    "is_active": True,
    "skills": ["Python", "SQL"]
}

2. Как создавать словари?

① Пустой словарь

empty_dict = {}
# или
empty_dict = dict()

② С предустановленными значениями

person = {
    "name": "Иван",
    "age": 30,
    "city": "Москва"
}

③ Из пар ключ-значение

keys = ['name', 'age']
values = ['Мария', 28]
person = dict(zip(keys, values))  # {'name': 'Мария', 'age': 28}

3. Как получать данные?

① Через ключ (вызовет ошибку если ключа нет)

print(person["name"])  # 'Иван'

② Метод get() (безопасный способ)

print(person.get("name"))      # 'Иван'
print(person.get("salary"))    # None (не ошибка)
print(person.get("salary", 0)) # 0 (значение по умолчанию)

③ Проверка наличия ключа

if "age" in person:
    print("Ключ 'age' существует")

4. Как изменять словари?

① Добавление/изменение элемента

person["job"] = "Программист"  # Добавит новый ключ
person["age"] = 31             # Обновит существующий

② Удаление элемента

del person["city"]          # Удаление по ключу
age = person.pop("age")     # Удаление + возврат значения
person.clear()              # Очистка всего словаря

5. Полезные методы

Метод Пример Описание
keys() person.keys() Все ключи
values() person.values() Все значения
items() person.items() Пары (ключ, значение)
update() person.update({"salary": 50000}) Добавить несколько пар
copy() new_dict = person.copy() Копия словаря

Пример перебора:

for key, value in person.items():
    print(f"{key}: {value}")

6. Генераторы словарей

Создание словарей в одну строку:

squares = {x: x**2 for x in range(5)}  # {0:0, 1:1, 2:4, 3:9, 4:16}

7. Примеры использования

① Телефонная книга

contacts = {
    "Иван": "+79123456789",
    "Мария": "+79098765432"
}

name = input("Введите имя: ")
print(f"Телефон: {contacts.get(name, 'не найден')}")

② Подсчет слов

text = "яблоко груша яблоко апельсин"
words = text.split()
count = {}

for word in words:
    count[word] = count.get(word, 0) + 1

print(count)  # {'яблоко':2, 'груша':1, 'апельсин':1}

8. Важные особенности

  • Ключи должны быть неизменяемыми (строки, числа, кортежи)
  • Порядок элементов сохраняется (начиная с Python 3.7)
  • Поиск по ключу происходит мгновенно (O(1))
  • Нельзя иметь дубликаты ключей

9. Частые ошибки

❌ Обращение к несуществующему ключу

print(person["salary"])  # KeyError

✅ Исправление:

print(person.get("salary", 0))  # Вернет 0 если ключа нет

❌ Использование изменяемых объектов как ключей

d = {[1,2]: "value"}  # TypeError

✅ Исправление:

d = {tuple([1,2]): "value"}  # Кортежи можно использовать

💡 Совет

Для сложных структур используйте вложенные словари:

users = {
    "user1": {"name": "Анна", "age": 25},
    "user2": {"name": "Борис", "age": 30}
}
print(users["user1"]["name"])  # 'Анна'
