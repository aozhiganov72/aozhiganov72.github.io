В Python сравнения выполняются с помощью операторов сравнения, которые возвращают True или False в зависимости от условия.

Основные операторы сравнения:

Оператор Описание Пример
== Равно 5 == 5 → True
!= Не равно 3 != 2 → True
> Больше 10 > 5 → True
< Меньше 3 < 1 → False
>= Больше или равно 7 >= 7 → True
<= Меньше или равно 4 <= 5 → True

Примеры:

a = 10
b = 20

print(a == b)  # False
print(a != b)  # True
print(a > b)   # False
print(a < b)   # True
print(a >= 10) # True
print(b <= 20) # True

Особенности сравнений в Python:

  1. Сравнение разных типов
    Python позволяет сравнивать разные типы, но результат может быть неочевидным (например, 1 == 1.0 → True, но '1' == 1 → False).

    print(1 == True)   # True (так как True == 1)
print(0 == False)  # True (так как False == 0)

  2. Цепочки сравнений
    Можно записывать несколько сравнений в одной строке:

    x = 5
print(0 < x <= 10)  # True (эквивалентно 0 < x and x <= 10)

  3. Сравнение строк
    Строки сравниваются лексикографически (по кодам Unicode):

    print("apple" < "banana")  # True ('a' < 'b')
print("cat" == "Cat")      # False ('c' != 'C')

  4. Сравнение коллекций (списков, кортежей)
    Сравнение происходит поэлементно:

    print([1, 2] < [1, 3])  # True (2 < 3)
print((1, 2) == (1, 2)) # True

  5. Оператор is (проверка идентичности объектов)
    Проверяет, ссылаются ли переменные на один и тот же объект в памяти:

    x = [1, 2]
y = [1, 2]
print(x == y)  # True (значения одинаковые)
print(x is y)  # False (разные объекты)

Проверка вхождения (in и not in):

Эти операторы проверяют наличие элемента в коллекции:

nums = [1, 2, 3]
print(2 in nums)     # True
print(5 not in nums) # True

Логические операторы (and, or, not):

Используются для комбинирования условий:

age = 25
print(age >= 18 and age <= 30)  # True
print(age < 18 or age > 60)     # False
print(not age == 25)            # False
