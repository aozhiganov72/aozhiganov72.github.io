–í Python —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç True –∏–ª–∏ False –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —É—Å–ª–æ–≤–∏—è.

–û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è:

–û–ø–µ—Ä–∞—Ç–æ—Ä –û–ø–∏—Å–∞–Ω–∏–µ –ü—Ä–∏–º–µ—Ä
== –†–∞–≤–Ω–æ 5 == 5 ‚Üí True

> –ë–æ–ª—å—à–µ 10 > 5 ‚Üí True
< –ú–µ–Ω—å—à–µ 3 < 1 ‚Üí False
>= –ë–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ 7 >= 7 ‚Üí True
<= –ú–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ 4 <= 5 ‚Üí True

–ü—Ä–∏–º–µ—Ä—ã:

a = 10
b = 20

print(a == b)  # False

–û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–π –≤ Python:

  1. –°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–æ–≤
    Python –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã, –Ω–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ–æ—á–µ–≤–∏–¥–Ω—ã–º (–Ω–∞–ø—Ä–∏–º–µ—Ä, 1 == 1.0 ‚Üí True, –Ω–æ '1' == 1 ‚Üí False).

    print(1 == True)   # True (—Ç–∞–∫ –∫–∞–∫ True == 1)

  2. –¶–µ–ø–æ—á–∫–∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–π
    –ú–æ–∂–Ω–æ –∑–∞–ø–∏—Å—ã–≤–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏–π –≤ –æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–µ:

    x = 5
print(0 < x <= 10)  # True (—ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ 0 < x and x <= 10)

  3. –°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫
    –°—Ç—Ä–æ–∫–∏ —Å—Ä–∞–≤–Ω–∏–≤–∞—é—Ç—Å—è –ª–µ–∫—Å–∏–∫–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏ (–ø–æ –∫–æ–¥–∞–º Unicode):

    print("apple" < "banana")  # True ('a' < 'b')

  4. –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –∫–æ–ª–ª–µ–∫—Ü–∏–π (—Å–ø–∏—Å–∫–æ–≤, –∫–æ—Ä—Ç–µ–∂–µ–π)
    –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ—ç–ª–µ–º–µ–Ω—Ç–Ω–æ:

    print([1, 2] < [1, 3])  # True (2 < 3)
print((1, 2) == (1, 2)) # True

  5. –û–ø–µ—Ä–∞—Ç–æ—Ä is (–ø—Ä–æ–≤–µ—Ä–∫–∞ –∏–¥–µ–Ω—Ç–∏—á–Ω–æ—Å—Ç–∏ –æ–±—ä–µ–∫—Ç–æ–≤)
    –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, —Å—Å—ã–ª–∞—é—Ç—Å—è –ª–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –Ω–∞ –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –æ–±—ä–µ–∫—Ç –≤ –ø–∞–º—è—Ç–∏:

y = [1, 2]

print(x is y)  # False (—Ä–∞–∑–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã)

–ü—Ä–æ–≤–µ—Ä–∫–∞ –≤—Ö–æ–∂–¥–µ–Ω–∏—è (in –∏ not in):

–≠—Ç–∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–æ–≤–µ—Ä—è—é—Ç –Ω–∞–ª–∏—á–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ –∫–æ–ª–ª–µ–∫—Ü–∏–∏:

–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã (and, or, not):

–ò—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –¥–ª—è –∫–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–∏—è —É—Å–ª–æ–≤–∏–π:
