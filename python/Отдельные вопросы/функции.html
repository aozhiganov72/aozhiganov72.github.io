Вот список наиболее употребимых функций в Python, которые часто используются в повседневном программировании:

1. Встроенные функции (Built-in Functions)

Эти функции доступны без импорта каких-либо модулей.

  • print() – вывод данных в консоль.
  • input() – чтение ввода пользователя.
  • len() – возвращает длину объекта (строка, список, кортеж и т. д.).
  • type() – возвращает тип объекта.
  • int(), float(), str(), bool() – преобразуют данные в соответствующий тип.
  • range() – создает последовательность чисел.
  • sum() – возвращает сумму элементов.
  • min(), max() – находят минимальный и максимальный элемент.
  • abs() – возвращает абсолютное значение числа.
  • round() – округляет число.
  • sorted() – возвращает отсортированный список.
  • reversed() – возвращает обратный итератор.
  • enumerate() – возвращает пары (индекс, значение).
  • zip() – объединяет элементы нескольких итерируемых объектов.
  • map() – применяет функцию к каждому элементу.
  • filter() – фильтрует элементы по условию.
  • any(), all() – проверяют, есть ли хотя бы один True или все True.
  • isinstance() – проверяет тип объекта.
  • dir() – возвращает список атрибутов объекта.
  • help() – выводит справку по объекту.
  • open() – открывает файл для чтения/записи.

2. Функции для работы со списками (List Methods)

  • .append() – добавляет элемент в конец списка.
  • .extend() – расширяет список другим списком.
  • .insert() – вставляет элемент по индексу.
  • .remove() – удаляет первое вхождение элемента.
  • .pop() – удаляет и возвращает элемент по индексу.
  • .index() – возвращает индекс элемента.
  • .count() – считает количество вхождений элемента.
  • .sort() – сортирует список.
  • .reverse() – разворачивает список.
  • .copy() – создает копию списка.
  • .clear() – очищает список.

3. Функции для работы со строками (String Methods)

  • .strip() – удаляет пробелы в начале и конце строки.
  • .split() – разбивает строку по разделителю.
  • .join() – объединяет список строк в одну строку.
  • .upper(), .lower() – преобразует строку в верхний/нижний регистр.
  • .capitalize() – делает первую букву заглавной.
  • .title() – делает каждое слово с заглавной буквы.
  • .replace() – заменяет подстроку в строке.
  • .find(), .index() – ищет подстроку в строке.
  • .startswith(), .endswith() – проверяет начало/конец строки.
  • .isdigit(), .isalpha(), .isalnum() – проверяют, состоит ли строка из цифр, букв или букв и цифр.

4. Функции для работы со словарями (Dict Methods)

  • .keys() – возвращает ключи словаря.
  • .values() – возвращает значения словаря.
  • .items() – возвращает пары (ключ, значение).
  • .get() – безопасное получение значения по ключу.
  • .update() – обновляет словарь.
  • .pop() – удаляет ключ и возвращает значение.
  • .setdefault() – возвращает значение, если ключ есть, иначе добавляет ключ.
  • .clear() – очищает словарь.

5. Функции для работы с множествами (Set Methods)

  • .add() – добавляет элемент в множество.
  • .remove(), .discard() – удаляют элемент.
  • .union() (или |) – объединение множеств.
  • .intersection() (или &) – пересечение множеств.
  • .difference() (или -) – разность множеств.
  • .symmetric_difference() (или ^) – симметричная разность.

6. Функции для работы с файлами

  • open() – открывает файл (режимы 'r', 'w', 'a', 'rb', 'wb' и др.).
  • .read() – читает содержимое файла.
  • .readline() – читает строку.
  • .readlines() – читает все строки в список.
  • .write() – записывает данные в файл.
  • .close() – закрывает файл.

7. Другие полезные функции (из стандартных модулей)

  • math.sqrt() – квадратный корень.
  • math.pow() – возведение в степень.
  • random.randint() – случайное целое число.
  • random.choice() – случайный элемент списка.
  • datetime.datetime.now() – текущая дата и время.
  • json.loads(), json.dumps() – работа с JSON.
  • os.listdir() – список файлов в директории.
  • os.path.join() – объединяет пути.

Этот список охватывает основные функции, которые чаще всего используются в Python. Если нужно углубиться в какую-то категорию, дай знать! 🚀
