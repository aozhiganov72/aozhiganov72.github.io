Цветной вывод в Python

В Python есть несколько способов выводить цветной текст в консоль. Вот основные методы:

1. Использование ANSI escape-кодов

# Базовые цвета
print("\033[31mКрасный текст\033[0m")
print("\033[32mЗеленый текст\033[0m")
print("\033[33mЖелтый текст\033[0m")
print("\033[34mСиний текст\033[0m")
print("\033[35mПурпурный текст\033[0m")
print("\033[36mГолубой текст\033[0m")

# Жирный текст
print("\033[1mЖирный текст\033[0m")

# Подчеркнутый текст
print("\033[4mПодчеркнутый текст\033[0m")

# Фоновые цвета
print("\033[41mКрасный фон\033[0m")
print("\033[42mЗеленый фон\033[0m")

2. Использование модуля colorama (кросс-платформенное решение)

from colorama import Fore, Back, Style, init

# Инициализация (для Windows)
init()

print(Fore.RED + 'Красный текст')
print(Back.GREEN + 'Зеленый фон')
print(Style.BRIGHT + 'Яркий текст')
print(Style.RESET_ALL + 'Обычный текст')

3. Использование модуля termcolor

from termcolor import colored

print(colored('Красный текст', 'red'))
print(colored('Зеленый текст на желтом фоне', 'green', 'on_yellow'))
print(colored('Жирный синий текст', 'blue', attrs=['bold']))

4. Использование модуля rich (продвинутое форматирование)

from rich.console import Console

console = Console()

console.print("Обычный текст")
console.print("[bold red]Жирный красный текст[/bold red]")
console.print("[blue on white]Синий текст на белом фоне[/blue on white]")
console.print(":smiley: Эмодзи и [u]подчеркнутый[/u] текст")

5. Использование модуля blessed (для продвинутого управления терминалом)

from blessed import Terminal

term = Terminal()
print(term.red('Красный текст'))
print(term.green('Зеленый текст'))
print(term.bold_blue_on_white('Жирный синий на белом фоне'))

Для большинства задач достаточно colorama или termcolor. Если нужны более сложные возможности форматирования, стоит выбрать rich.
