День 1: Установка Python и первая программа

Цель:
- Убедиться, что Python установлен и работает.
- Написать и запустить первый код.
- Понять базовый синтаксис (отступы, комментарии).

1. Установка Python (если ещё нет)

Проверяем текущую версию Python:

Открой терминал в Linux и введи:

python3 --version

Если Python установлен, увидишь что-то вроде:

Python 3.10.12

Если Python не установлен:

Введи в терминале:

sudo apt update && sudo apt install python3 -y

Дополнительно: Установка pip (менеджер пакетов)

sudo apt install python3-pip -y

2. Установка редактора кода (Geany)

Поскольку ты работаешь в Linux и предпочитаешь Geany, установим его:

sudo apt install geany -y

Настройка Geany для Python:

  1. Открой Geany.
  2. Перейди в Создать → Новый файл (или Ctrl+N).
  3. Сохрани файл как hello.py (Ctrl+S).
  4. В меню Сборка → Установить команды сборки выбери Python:
    • Команда на выполнение: python3 "%f"

Теперь можно запускать код кнопкой F5.

3. Первая программа: Hello, World!

Открой hello.py и напиши:

#!/usr/bin/env python3  # Указываем, что скрипт нужно запускать через Python 3
print("Hello, World!")  # Выводим текст

Как запустить?

Способ 1 (через Geany):
- Нажми F5 – результат появится в нижней консоли.

Способ 2 (через терминал):

python3 hello.py

Результат:

Hello, World!

Дополнительно: Сделать скрипт исполняемым

Если хочешь запускать файл как программу (например, ./hello.py):

chmod +x hello.py  # Даём права на выполнение
./hello.py         # Запускаем

4. Основы синтаксиса

Комментарии

# Это однострочный комментарий
print("Это код")  # Комментарий после строки

"""
А это многострочный комментарий
(на самом деле — строка, но её никто не использует)
"""

Отступы (важно!)

В Python блоки кода выделяются отступами, а не скобками {}.

if 5 > 2:
    print("Пять больше двух!")  # ← 4 пробела или Tab
    print("Это внутри условия")  
print("А это уже снаружи")

Ошибка:

if 5 > 2:
print("Так нельзя!")  # ← SyntaxError: expected an indented block
